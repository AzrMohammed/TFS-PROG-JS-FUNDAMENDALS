# Problem Statement   

By default initialize a variable as "myMark" with integer value.            
Write an algorithm to print the suitable grade for the give mark


## Rules

    | Mark | Grade |  
    | :---:   |  :----:   | 
    | 91 - 100 | A | 
    | 81 - 90 | B | 
    | 71 - 80 | C | 
    | 61 - 70 | D | 
    | 51 - 60 | E | 
    | 0 - 50 | F | 
    | > 100 | Invalid Mark |
