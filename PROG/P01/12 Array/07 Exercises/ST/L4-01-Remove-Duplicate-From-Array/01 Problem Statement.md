# Problem Statement   

Write a JavaScript program to remove duplicate items from an array.

## Rules

* Output array should not contain duplicate for Ex: array =[12,33,33,4,0,4]
    * Print [12,33,4,0]

## Sample I/O

> #### Sample Input 1:
> array = [1,2,3,1,2]
>
> ##### Output:
> [1,2,3]

> #### Sample Input 2:
> array = [4,5,3,3,4,0] 
>
> ##### Output:
> [4,5,3,0]

> #### Sample Input 3:
> array = [1,1,30,100,20,33,33]
>
> ##### Output:
> [1,30,100,20,33]
